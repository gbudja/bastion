# Security Policy

## Supported Versions

Currently supported versions:

| Version | Supported |
|--------|----------|
| 0.2.x  | Yes      |

## Reporting a Vulnerability

If you discover a security vulnerability, please do not open a public issue.

Instead:
- Email: grantbobo31@gmail.com
- Or open a private security advisory via GitHub

Please include:
- description of the issue
- steps to reproduce
- potential impact

## Notes

Bastion is a security-focused project. Responsible disclosure is appreciated.
